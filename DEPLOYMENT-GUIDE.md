## Quick Deploy (3 Steps)

### Step 1: Extract the Files
1. Download the `lwv-newport-county-website-COMPLETE.tar.gz` file
2. Extract it (double-click on Mac/Windows, or use `tar -xzf filename.tar.gz`)
3. You'll have a folder with all files

### Step 2: Upload to GitHub
1. Go to https://github.com
2. Click "New repository"
3. Name it: `lwv-newport-county`
4. Make it **Public**
5. Click "Create repository"
6. Click "uploading an existing file"
7. **Drag ALL files** from your extracted folder (including the images folder!)
8. Commit the files

### Step 3: Enable GitHub Pages
1. In your repository, click **Settings**
2. Click **Pages** in the left sidebar
3. Under "Source", select **main** branch
4. Click **Save**
5. Wait 1-2 minutes

**🎉 Your site is live at:**
`https://YOUR-USERNAME.github.io/lwv-newport-county/`

## Custom Domain (Optional)

Want to use your own domain like `www.lwvnewportcounty.org`?

1. Buy a domain (from Namecheap, Google Domains, etc.)
2. In GitHub Settings → Pages, add your custom domain
3. Update your domain's DNS settings (GitHub provides instructions)

## Troubleshooting

**Site not showing up?**
- Wait 2-3 minutes after enabling Pages
- Make sure repository is Public
- Check that `index.html` is in the root folder

**Photos not showing?**
- Make sure `images/` folder was uploaded
- Check that image filenames match exactly

**Links not working?**
- Make sure all HTML files were uploaded
- Check that navigation links are correct
